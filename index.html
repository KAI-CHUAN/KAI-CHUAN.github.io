網際網路　檔案館


https://archive.org/



　　網際網路檔案館（英語：Internet Archive）是一個非營利性的數位圖書館。成立於1996年，由Alexa創始人布魯斯特·卡　利創辦。提供數字資料如網站、音樂、動態圖像、和數百萬書籍的永久性免費儲存及獲取。迄至2012年10月，其資訊儲量達到10PB。除此之外，該檔案館也是網路開放與自由化的倡議者之一。
　　其資料是由自帶的網路爬蟲自動搜集的，其網頁檔案館「時光機」（Wayback Machine），抓取了超過1500億的網頁。
　　其年度預算約100萬美元，來源則是其網頁爬蟲服務、合作關係、贊助以及卡利-奧斯丁基金會（Kahle-Austin Foundation）。總部位於舊金山里奇蒙德區，此地僱員只有數十人，大部份僱員工作於書籍掃描中心，在紅木城也有資料中心。
　　該資料庫是國際網際網路儲存聯盟成員，2007年被加利福尼亞州選為官方指定圖書館。
　　你可以在這裡找到各式各樣網頁的歷史頁面！

相片詳細資料讀取


http://regex.info/exif.cgi


　　可交換圖檔格式常被簡稱為Exif（Exchangeable image file format），是專門為數位相機的相片設定的，可以記錄數位相片的屬性資訊和拍攝資料。
　　最初由日本電子工業發展協會在1996年制定，版本為1.0。1998年，升級到2.1，增加了對音訊檔案的支援。2002年3月，發表了2.2版。
　　Exif可以附加於JPEG、TIFF、RIFF等檔案之中，為其增加有關數位相機拍攝資訊的內容和索引圖或圖像處理軟體的版本資訊。
　　Windows 7作業系統具備對Exif的原生支援，通過滑鼠右鍵點擊圖片開啟選單，點擊屬性並切換到詳細資訊標籤下即可直接檢視Exif資訊。
　Exif資訊是可以被任意編輯的，因此只有參考的功能。Exif資訊以0xFFE1作為開頭標記，後兩個位元組表示Exif資訊的長度。所以Exif資訊最大為 64 kB，而內部採用TIFF格式。
　但若想讀到照相的相機、鏡頭、曝光時間、光圈、快門等資料往往就需要比較專業的軟體幫忙。這裡附上一位網友寫的線上版，除了若檔案很大的話需要傳比較久外，這個服務真的相當方便。
